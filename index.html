О компании

Zonlik Fabric — это независимая разработка инструментов для системного администрирования и мониторинга.
Мы создаём лёгкие, быстрые и информативные программы для Linux, которые помогают пользователям старых и слабых ПК.
Наша цель — дать каждому возможность контролировать своё железо без лишних затрат ресурсов.

📊 Zonlik Monitor

Системный монитор для Linux

📊 Zonlik Monitor

Версия: 2.0 | Платформа: Linux | Лицензия: Open Source

Лёгкий и быстрый системный монитор, созданный специально для старых ноутбуков и слабых ПК. Позволяет отслеживать загрузку системы, находить узкие места и оптимизировать работу.

🔧 Возможности

📊 Мониторинг

Графики CPU, RAM, GPU, диски, сеть. Кнопки вкл/выкл под каждым графиком.

🔄 Процессы

Список всех процессов. Убийство выбранного. Топ-5 жирных процессов.

💻 Процессор

Модель, ядра, частота, температура, флаги SSE/AVX/AVX-512.

💾 Оперативная память

Объём, загрузка, SWAP. Детали через dmidecode.

🎮 Видеокарта

nvidia-smi: модель, драйвер, температура, память.

🔋 Батарея

Заряд, износ, калибровка, график истории.

💿 Диски

Список дисков, SMART, температура, разделы.

📀 USB

Список устройств, диагностика, форматирование.
