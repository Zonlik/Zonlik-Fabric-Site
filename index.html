Системный монитор для Linux | v2.0

Лёгкий, быстрый, информативный

🐧 Linux ⚡ Низкое потребление 📊 Графики 🔋 Мониторинг батареи
Zonlik Fabric — создан для старых ноутбуков и слабых ПК. Помогает понять, что именно тормозит систему: CPU, RAM, диск или перегрев.
📊 Мониторинг
CPU, RAM, GPU, диски, сеть — графики и цифры
🔋 Батарея
Заряд, износ, калибровка, график истории
💾 Диски
SMART, температура, скорость, разделы
🎮 Видеокарта
nvidia-smi, температура, память, частота
🔄 Процессы
Список, убийство, топ-5 жирных процессов
⚙️ Настройки
Темы, стили графиков, автоочистка RAM
💻 Процессор
Модель, частота, температура, флаги
📀 USB
Список устройств, диагностика
📥 Скачать Zonlik Fabric
🚀 Быстрый запуск (одной командой):
python3 zonlik_monitor_full.py
🌐 Запуск через curl (без скачивания):
python3 <(curl -s https://zonlik.github.io/Zonlik-Fabric-Site/zonlik_monitor_full.py)
📦 Требования:
• Linux (Ubuntu, Mint, Debian, Fedora, Arch)
• Python 3.8+
• Установка зависимостей: pip install psutil
🖼️ Иконка на рабочий стол:
cat > ~/Рабочий\ стол/zonlik-fabric.desktop << EOF
[Desktop Entry]
Type=Application
Name=Zonlik Fabric
Exec=python3 /home/kirill/Zonlik-Monitor/zonlik_monitor_full.py
Icon=/home/kirill/Zonlik-Monitor/icon.png
Terminal=false
Categories=System;
EOF
