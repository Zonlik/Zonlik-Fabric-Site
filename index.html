❓ Часто задаваемые вопросы

❓ Программа не запускается
Установите Python 3.8+ и библиотеку psutil: pip install psutil
❓ GPU показывает --%
У вас драйвер nouveau. Установите nvidia-driver-390 для полной поддержки.
❓ Как добавить в автозапуск?
В настройках программы поставьте галочку "Автозапуск при старте системы".
❓ Графики дисков и сети пустые
Подождите 30-60 секунд — данные накапливаются.
❓ Как очистить кэш?
Нажмите кнопку "Очистить кэш" на вкладке "Мониторинг".
