❓ Часто задаваемые вопросы

❓ Программа не запускается
Установите Python 3.8+ и psutil: pip install psutil
❓ GPU показывает --%
Установите nvidia-driver-390 для полной поддержки.
❓ Как добавить в автозапуск?
В настройках программы поставьте галочку "Автозапуск".

📥 Установка

  1. Скачайте ZIP-архив или скрипт
  2. Распакуйте архив
  3. Установите зависимости: pip install psutil
  4. Запустите: python3 zonlik_monitor_full.py
🚀 Быстрый запуск (curl):
python3 <(curl -s https://zonlik.github.io/Zonlik-Fabric-Site/zonlik_monitor_full.py)
